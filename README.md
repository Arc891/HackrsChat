# HackrsChat

HackrsChat is a personal-use self-host TUI chat application for you and your friends/community written in Rust with use of the Cursive library. 
This project is currently a work in progress, feel free to contribute!

## Roadmap
- [ ] Basic TUI layout
- [ ] Basic chat functionality
- [ ] User authentication
- [ ] User registration
- [ ] Basic chat commands
- [ ] More to come...
